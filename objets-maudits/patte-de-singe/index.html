Patte de singe

Objet maudit a fort levier strategique: offre des effets majeurs (resurrection, info, meteo, fuite), mais chaque voeu impose une contrepartie lourde.

Difficulte: Tres elevee Cout sanite: Variable selon voeu (peut inclure gros malus) Risque chasse: De moyen a critique selon le voeu Limite: 3 a 5 voeux selon multiplicateur de partie

Infos rapides

  • Difficulte d'usage: Tres elevee
  • Cout principal: Variable selon voeu (peut inclure gros malus)
  • Risque de chasse: De moyen a critique selon le voeu
  • Limite d'utilisation: 3 a 5 voeux selon multiplicateur de partie

Utilise ces objets uniquement avec plan defensif actif (cache valide, sortie, encens pret).

Procedure d'utilisation

  • Tenir la patte et formuler un voeu valide.
  • Verifier immediatement l'effet obtenu puis le malus applique.
  • Planifier les voeux restants: un meme voeu ne peut pas etre repete.

Risques majeurs

  • Peut lancer une chasse maudite immediate ou differree selon le voeu.
  • Peut imposer perte severe de sanite, handicap visuel/audio ou mort directe.
  • Le nombre de voeux baisse avec le multiplicateur de difficulte.

Pro tips

  • Garder les voeux critiques (fuite, resurrection) pour les moments de crise.
  • Eviter les voeux agressifs sans setup defensif deja en place.
  • En equipe, designer un seul porteur pour suivre les contreparties.
