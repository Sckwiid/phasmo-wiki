## Deploiement GitHub Pages

1. Pousser ce repository sur GitHub.
2. Dans `Settings > Pages`, choisir la branche (`main`) et le dossier racine (`/root`).
3. Verifier le domaine final puis adapter les URLs canoniques/sitemap si necessaire.

## SEO

- Pages en dossiers avec `index.html`.
- `robots.txt` + `sitemap.xml`.
- Meta tags (title/description/keywords/og) par page.
- JSON-LD sur les pages principales.
