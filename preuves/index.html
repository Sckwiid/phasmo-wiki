Recoupe toujours avec la page Fantomes pour confirmer via un comportement exclusif, surtout en difficulte reduisant les preuves.

Conseils de verification fiable

  • Place plusieurs outils simultanement dans la piece hantee pour eviter les faux negatifs.
  • Donne 30 a 60 secondes entre deux tests actifs (Spirit Box, DOTS, ecriture).
  • Filme DOTS et orbes en continu depuis le camion pour conserver une preuve visuelle stable.
  • Avec UV, teste juste apres interaction (porte, interrupteur, fenetre, sel).
