Transparence des donnees

Methodologie de construction du wiki

Le contenu de ce site est construit a partir des fichiers HTML locaux fournis, puis reformule et structure en pages thematiques pour la recherche SEO et l'usage en mission.

Comment les donnees sont croisees

  • Extraction des tableaux entites/preuves et equipements.
  • Recoupement avec mecaniques de sanite, cartes et comportements.
  • Normalisation dans une base JS unique exploitee par toutes les pages.
  • Reformulation en francais pour eviter le copier-coller brut.

Contraintes techniques

  • Site statique pur (HTML/CSS/JS), compatible GitHub Pages.
  • URLs propres via dossiers (`/page/`) pour un meilleur referencement.
  • Sitemap + robots + maillage interne + metadonnees par page.
  • Design sombre minimaliste, lisible sur desktop et mobile.
